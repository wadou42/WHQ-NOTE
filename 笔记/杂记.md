### jmeter
```bash
jmeter -n -t <测试计划文件路径> -l <结果文件路径> -e -o <报告输出目录>

eg:
jmeter -n -t F:\apache-jmeter-5.6.3\apache-jmeter-5.6.3\work_space\"View Results Tree.jmx" -l F:\apache-jmeter-5.6.3\apache-jmeter-5.6.3\work_space\out.jtl -e -o F:\apache-jmeter-5.6.3\apache-jmeter-5.6.3\work_space\report.txt
```
